# Security Policy

## Supported Use

This repo is meant for local, manually authenticated Gemini web-session automation.

## Please Do Not Report

- Requests for cookie import support
- Requests for login bypasses
- API billing bypass attempts

## Please Report

- Leaks of local machine paths or personal data in docs
- Unsafe logging of session-sensitive browser data
- Accidental persistence of auth material beyond normal browser cookies
- Any file write path that can escape the intended output location
